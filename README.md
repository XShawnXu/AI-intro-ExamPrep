# 《人工智能导论》刷题系统 (AI-intro-ExamPrep)

- **题库为本学期（26春）课程群中老师发布的题库，原文复制，题目信息准确。** 
- 一个单文件、纯前端的在线刷题网页。 没有任何后端和依赖，为人工智能导论期末备考制作，开箱即用。
## ✨ 主要功能

- **纯前端单文件**：只有一个 `index.html`，零配置。
- **本地错题本**：做错的题自动记入本地缓存（LocalStorage），重刷答对后自动移除。
- **键盘快捷键**：电脑端支持按 `A / B / C / D` 选答案，`Enter` 提交/下一题。
- **移动端适配**：底部固定悬浮按钮，手机单手刷题体验良好。
- **两种模式**：支持按章节/题组顺序刷题，以及全库随机抽题模拟考。

## 🚀 怎么使用？

**在线使用：**
- [点击此处 - 人工智能导论刷题系统](https://xshawnxu.github.io/AI-intro-ExamPrep/)
- **在线使用同样可以保留错题记录**

**本地使用：**
- 直接下载项目里的 `index.html`，双击用浏览器打开即可。
